# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

User: The person using the application to create, organize, review, update, and delete ideas in their personal vault. 

## Draft User Stories

1. As a user, I want to view all of my saved ideas so that I can see everything stored in my vault. 

2. As a user, I want to add a new idea so that I can save it before I forget it. 

3. As a user, I want to assign a category to an idea so that I can organize similar ideas together. 

4. As a user, I want to assign a risk level to an idea so that I can judge how dangerous or unrealistic it may be.

5. As a user, I want to enter an estimated cost for an idea so that I can decide whether it is affordable. 

6. As a user, I want to edit an existing idea so that I can update its details later. 

7. As a user, I want to delete an idea so that I can remove ideas I no longer want to keep. 

8. As a user, I want to mark an idea as a favorite so that I can easily return to my best ideas. 

9. As a user, I want to add notes about possible benefits and problems so that I can evaluate an idea more carefully. 

10. As a user, I want to view a random idea from my vault so that I can rediscover an idea I may have forgotten.
