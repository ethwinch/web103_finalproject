# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1: Collector: a user who is cultivating the media in the library.
2: Lurker: a user who is only there to browse the collections of Collectors. 

## Draft User Stories

[👉🏾👉🏾👉🏾 Include at least at least 10 user stories in this format:]

1: As a netizen of the indie web, I want sites that help me share my passions in order to better connect with other netizens. 
2: As a book lover, music enjoyer, and movie buff, I want a place to track all the media I watch to look back on and remember while sharing my current tastes with others.
3: As a person in fandom, I want a site to display all the shows and fandoms I’m into so I can find others with similar interests and form a community.
